# PyBrowser - A Simple Web Browser in Python

A lightweight web browser built using Python and PyQt5 with QtWebEngine.

## Features

- **Navigation Controls**: Back, forward, reload, and home buttons
- **URL Bar**: Direct URL input with auto-completion of 'https://'
- **Keyboard Shortcuts**:
  - `Alt+Left`: Go back
  - `Alt+Right`: Go forward
  - `F5`: Reload page
  - `Ctrl+H`: Go to homepage
- **Modern Web Support**: Built on QtWebEngine for full web compatibility
- **Cross-Platform**: Works on Windows, macOS, and Linux
